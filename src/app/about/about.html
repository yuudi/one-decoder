若有代码问题请提交 Issue，非代码问题也欢迎参与自由讨论

快捷链接

可以将需要解码的文本直接放在 URL 中，例如
{{ baseUrl }}#/decoder?input=57uf5LiA6Kej56CB

PWA

本站支持渐进式 Web 应用（PWA），用户可以将本页面安装为应用，离线使用。

PWA 安装方法可参考 Google Web Dev

安装为 PWA 后，可以直接将文本分享给本应用，获取解码结果。（需要系统支持，目前支持情况请参考 MDN ）

本站不收集任何用户数据，所有解码均在本地浏览器完成，用户的输入、输出不会被发送到任何服务器。

本站不使用任何分析服务（如 Google Analytics）或第三方 Cookie。

本站托管于 GitHub Pages ，基础资源由 GitHub 的服务器处理。
